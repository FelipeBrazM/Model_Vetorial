As pontuações de dois atletas em um campeonato são representados em duas matrizes de números inteiros, uma para cada atleta. O campeonato é composto por provas, e cada prova é composta de várias etapas. Desejamos saber quantas vezes cada atleta ganhou (obteve pontuação maior) e quantos empates ocorreram. Implemente um programa que faça a leitura das duas matrizes de pontuações, uma para cada atleta. Nestas matrizes, cada linha representa uma prova e cada coluna representa uma etapa, e os valores representam as pontuações. Em seguida, realize o cálculo das vitórias de cada atleta e o número de empates, ao final, imprima os resultados no terminal. O programa deve verificar se as matrizes são compatíveis (possuem as mesmas dimensões).
